# dcsitalia-tools

A collection of **DCS World** utilities for the **DCORE** zone system (DCS Italia campaign). Tools for configuring, exporting, and validating links between trigger zones, used by modules such as DMAP.

## Contents

| Tool | Description |
|------|-------------|
| **DZONE_LINK_EDITOR** | Mission Editor extension to define zone links visually and export them to Lua. See [DZONE_LINK_EDITOR/README.md](DZONE_LINK_EDITOR/README.md). |
| **DZONE_TEST** | In-mission script to validate zone links on the F10 map during flight. |
